Evaluation

After writing your code, you can simply run python in the same folder as your code, then within the REPL do from exercises import * --- you can now test your functions by calling them directly from the REPL. E.g., see the following session log:

>>> from exercises import *
>>> times_table(3, 9)
[(3, 1, 3), (3, 2, 6), (3, 3, 9), (3, 4, 12), (3, 5, 15), (3, 6, 18), (3, 7, 21), (3, 8, 24), (3, 9, 27)]
>>> piglatinify('Hello there')
'ellohay erethay'
>>> quit()

Note that if you make changes to your file you will need to restart the REPL to test your modifications. As an alternative, you can manually reimport your module and use it as follows:

>>> from importlib import reload
>>> import exercises
>>> exercises.avg_attempts(100)
1.15
>>> # make some changes to the file
>>> reload(exercises)
<module 'exercises' from './cs331/mps/midterm01/exercises.py'>
>>> exercises.avg_attempts(100)
0.92
>>> quit()
