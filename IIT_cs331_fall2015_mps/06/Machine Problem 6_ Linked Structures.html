Overview

For this machine problem you will complete the implementation of the circular, doubly-linked list ADT started during class. You will need to flesh out the following methods, found stubbed out in the linkedlist.py file:

  • clear
  • count_if
  • extend
  • insert
  • pop
  • remove
  • __add__
  • __contains__
  • __delitem__
  • __getitem__

To support negative indexes (such as those supported by the built-in list), we've added the following function to the LinkedList class:

def __normalize_index(self, idx):
    """Normalizes negative indexes to positive values"""
    if idx < 0:
        idx = idx + self.count
        if idx < 0:
            idx = 0
    return idx

Note that calling it with a negative index value will return a properly normalized, positive index.

For more details on the __xxx___ methods you are to implement -- i.e., beyond the provided docstrings -- be sure to check out the emulating container types section of the Python documentation.
