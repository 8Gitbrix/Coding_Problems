Machine Problem 4: Array-backed List ADT

Objectives

  1. Implement an array-backed list abstract data type (ADT)
  2. Practice using Python's object oriented features
  3. Learn how to support Python's built-in sequence operators

Reminder: Before starting ...

... you'll need to do (git fetch upstream ; git merge upstream/master) within your cloned repository to fetch my additions (consisting of template code and recent lecture examples).

Overview

For this machine problem you will implement a list ADT, comprising most of the list API we've been using over the past few weeks. As the backing store for your implementation you will use Python's builtin list, restricted to just the following operations:

  • indexing; i.e., lst[i] and lst[i] = x
  • obtaining the length; i.e., len(lst)
  • appending an item to the end; i.e., lst.append(x)
  • deleting an item from the end; i.e., del lst[i]

Implementation Details

You will find template files for your implementation in the mps/04/ folder of your code repository. There are two files provided:

  1. arraylist.py, which contains the start of the list ADT implementation (completed during lecture). You will add methods to the ArrayList class contained therein to implement the API described below.
  2. main.py,

The methods you add to the ArrayList will support the following operations (given that lst, lst1, and lst2 is an instance of ArrayList):

  1. x in lst
  2. lst1 + lst2 (which returns a new instance of an ArrayList)
  3. len(lst)
  4. lst.count(x)
  5. del lst[i]
  6. lst.clear()
  7. lst1.extend(lst2)
  8. lst.insert(i, x)
  9. lst.remove(x)
  10. lst.pop([i])

See https://docs.python.org/3.5/library/stdtypes.html#common-sequence-operations and https://docs.python.org/3.5/library/stdtypes.html#mutable-sequence-types for a description of the above operations.
