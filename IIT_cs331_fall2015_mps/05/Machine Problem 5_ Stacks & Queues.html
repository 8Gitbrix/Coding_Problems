Machine Problem 5: Stacks & Queues

Objectives

  1. Use the Stack ADT
  2. Add to the circular array-backed Queue ADT
  3. Use the Queue ADT

Reminder: Before starting ...

... you'll need to do (git fetch upstream ; git merge upstream/master) within your cloned repository to fetch my additions (consisting of template code and recent lecture examples).

Overview

For this machine problem you will:

  • modify the check_parens function we started during lecture so that it checks for different types of paired delimiters (square brackets, angle brackets, and curly braces in addition to parentheses)
  • update the circular array-backed queue implementation so that its underlying array automatically expands when filled.

Implementation Details

You will find template files for your implementation in the mps/05/ folder of your code repository.

Finishing Up

When you're done, you'll need to:

  1. Commit your work (remember, you can do this frequently while working on the machine problem) with the command git commit -am 'Commit message'

  2. Push your changes to BitBucket with git push origin master

Check your BitBucket page in the browser to make sure your changes were pushed to the site.
